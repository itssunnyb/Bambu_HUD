# Bambu HUD

A standalone ESP32 touchscreen dashboard for Bambu Lab printers.

## Features

* Real-time printer monitoring
* AMS status display
* Touchscreen interface
* Web-based setup portal
* Multi-printer support
* Battery support (4" version)
* Compact 2.8" CYD support

## Supported Hardware

### 4" ESP32 Display

* 480x320 Touchscreen
* Battery Support

### 2.8" CYD

* ESP32-2432S028
* 320x240 Touchscreen

## Supported Printers

* P1S
* P1P
* X1C (Testing)
